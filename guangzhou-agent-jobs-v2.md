## 先说清楚：你的轨道不需要算法

AI 行业有两条完全不同的职业轨道：

```
算法工程师                          AI 应用开发工程师（你在这一侧）
───────────────────────────         ───────────────────────────
训练模型、调参、看论文               基于 API 搭 Agent、写业务逻辑、工程落地
PyTorch 底层、数学公式               Python、Tool Calling、RAG、LangChain
硕士起步，卷 985                     本科足够，项目经历 > 学历
35K-80K                             10K-30K（起步），天花板同样高
❌ 你不是这条路的                    ✅ 你已经在走这条路了
```

下面 20 家企业，**没有一家要求你会训练模型**。你手写 agent.py 练出来的能力——调 API、管理 messages 数组、处理 Tool Calling 循环、写 system prompt——正是这些岗位要的。
